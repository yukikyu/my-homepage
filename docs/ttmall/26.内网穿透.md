# 26.内网穿透

## 简介

内网穿透功能可以允许我们使用外网的网址来访问主机

正常的外网访问需要访问我们的项目的流程是：

1. 买服务器并且有公网固定IP
2. 买域名映射到服务器IP
3. 域名需要进行备案和审核

## 使用场景

1. 开发测试（微信、支付宝）
2. 智慧互联
3. 远程控制
4. 私有云
